# 1. Arquitetura do Sistema e Especificações Técnicas

## 1.1. Visão Geral da Arquitetura

O sistema será composto por uma API RESTful desenvolvida em Node.js com TypeScript e um Front-end Single Page Application (SPA) em React com TypeScript. A arquitetura do Back-end seguirá os princípios da **Clean Architecture** (Arquitetura Limpa) e **Domain-Driven Design (DDD)** simplificado, isolando as regras de negócio das tecnologias externas.
